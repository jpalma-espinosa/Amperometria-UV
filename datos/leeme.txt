Sigan el siguiente tutorial para instalar el programa Igor Pro y los macros correspondientes para
analizar las amperometrias 


1) Instale elprograma ejecutando el archivo setup que se encuentra en la carpeta IGOR PRO

2) Se abrirá el instalador, acepte las condiciones y terminos llegara el punto en el que se le pedirá
 la seria y clave de activacion:

NAME: IGOR PRO

SERIAL Number: 22088  install Kye: TJLSCENY

3) Terminada la instalación vaya a la carpeta de raíz donde se instalo el programa. 

Equipo disco-> disco C -> achivos de programa 86x -> WaveMetrics -> Igor Pro Folder -> Igor Procedures.

4) Una vez allí copie los archivos que se encuentran dentro de la carpeta macros del instalador y pegelos en 
direccion anterior. 


5) Vaya a la carpeta registros  ya abra sus archivos de formato Igor Packed Experiment File.
